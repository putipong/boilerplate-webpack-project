    <h1>Boilerplate Webpack project</h1>
</div>


<h3>Installation</h2>

```bash
git clone https://github.com/putipong/boilerplate-webpack-project.git
npm i
```

<h3>Usage</h4>

Build for local testing and development:
```bash
npm run dev
```

Bundle project for production:
```bash
npm run prod
```
